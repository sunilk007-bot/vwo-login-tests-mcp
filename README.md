# VWO Login Tests using MCP

This repository contains prompts and tasks for testing the VWO (Wingify) login page using Playwright MCP and Selenium MCP.

## Tasks

### Task 1 - Playwright MCP
Open https://app.vwo.com/ and enter random email and password, click signin, verify error message using Playwright MCP.

### Task 2 - Selenium MCP
Open https://app.vwo.com/ and enter random email and password, click signin, verify error message using Selenium MCP.

### Task 3 - GitHub MCP
Push all created code to a new GitHub repository using GitHub MCP.
